I'm a VMWare System Administrator based in Hawaii with 4 years of progressively increasing experience and responsibility in Windows and VMware. I have demonstrated experience in root cause analysis and troubleshooting complex environments with multiple dependencies (DNS, DHCP, PKI, AD, GPO).

A key theme throughout my career has been my knack for 1) identifying bottlenecks and 2) eliminating the bottlenecks, usually with automation. In my very first Helpdesk job at an MSP, a common task that was done manually was to install a clean Windows 10 OS, install updated drivers, and install our standard suite of common software (Office 365, RMM agent, anti-virus agent) on client computers. This was something I automated away by doing the initial legwork of configuring an MDT/WDS deployment server to image our client computers, which saved hundreds of hours of manual effort a month.

This has been a consistent theme later on with the use of Powershell and PowerCLI to script Windows OS patch installations, which reduced the time between the release of patches on Patch Tuesdays and the functional testing in our Testing lab, so patches could more quickly be released into Production. This later evolved into using Powershell Desired State Configuration for patch orchestration so that the process is fully automated and that services are always available (So Exchange 01 node will patch, and Exchange 02 will wait until the first node is finished before starting its own patching).

This knack of identifying and eliminating bottlenecks embodies good Systems Administration principles and makes me a force multiplier even within IT, which itself is a force multiplier within any larger organization. Combined with an “automation first” mindset, this tends to make me a key contributor in any roles I’ve held.
